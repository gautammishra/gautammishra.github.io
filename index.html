ReturnAtlas — Investment Analytics System

Independent Technical Project

ReturnAtlas is a production-grade investment analytics system I designed and built independently to explore deterministic backtesting, portfolio simulations, and risk-adjusted return computations with high correctness and operational discipline.

Over the course of the project, I took the system from architecture through ongoing operations, making deliberate tradeoffs between data quality, cost, and scalability. The core design emphasizes reproducible results and maintainability rather than superficial feature breadth.

Key features of the system include:

  • End-to-end architecture for financial data ingestion, normalization, and analytics, designed to scale from hundreds to thousands of symbols with minimal re-architecture.
  • Batch-first, idempotent data pipelines implemented with AWS Lambda, S3, SQS, and PostgreSQL, ensuring correctness even in the face of partial failures.
  • Stable REST APIs built with Spring Boot exposing deterministic backtesting (lump-sum and dollar-cost averaging), risk-adjusted returns (CAGR, IRR, Sharpe ratio, drawdown), and profile metrics.
  • Infrastructure as code using Terraform with strict environment isolation, IAM least-privilege, and automated deployment guardrails.
  • Clear architectural tradeoffs (e.g., prioritizing correctness and explainability over real-time streaming) documented and embodied in code.
